This page is only used to show the effect of using storage functions in JavaScript. The example is adapted from this Youtube tutorial.
Things you can do:

  1. Start the node server using: node storageserver.js
  2. Open the browser developer tool
  3. Go to the Application Tab in the developer tool
  4. Uncomment the code for the first run
  5. Restart the node server
  6. Check the Application Tab in the developer tool again, in particular local/session/cookie
  7. Now close the window, *comment* the code for the first run, and *uncomment* the code for the first run. You should be able to see that the session storage is gone, while the local storage is persistent. The persistence of your cookie will depend on the expiry data provided
